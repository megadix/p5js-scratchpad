# p5js-scratchpad

A (semi)random collection of [P5.js](https://p5js.org/) scripts: exercises, experiments... fun!

Please don't take these examples too seriously, as I am by no means a P5.js expert.

# Quickstart

You can either:

- open individual .html files
- (best option) serve files with a local http server

Example:

- install `serve`: a minimal and straightforward HTTP server written in Node
```
npm install -g serve
```
- `cd` into sources root
- start serving local files:
```
serve
```
- access pages:  
  http://localhost:5000
